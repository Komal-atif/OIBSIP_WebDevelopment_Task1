# OIBSIP - Web Development - Task 1: Interactive Landing Page

## Project Overview

This project is a responsive, interactive landing page for **REACT.gg** - a fictional platform that teaches React through gamified learning. The design features a vertical timeline with 11 learning levels, each represented by interactive nodes and content panels.

## Features

- **Animated Background**: Floating particle animations with gradient effects
- **Interactive Timeline**: 11 learning nodes with hover and click effects
- **Responsive Design**: Works on all devices (mobile, tablet, desktop)
- **Smooth Animations**: CSS transitions and JavaScript scroll effects
- **Modern UI/UX**: Clean design with consistent color scheme
- **Interactive Elements**: Clickable nodes, buttons, and video preview

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Advanced styling with Flexbox, Grid, and animations
- **JavaScript**: Interactive functionality and scroll effects
- **Font Awesome**: Icon library
- **Google Fonts**: Inter font family
- **CSS Variables**: Consistent color scheme management
